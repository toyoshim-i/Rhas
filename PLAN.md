# Rhas 今後の改善・拡張計画 (PLAN)

Rhas はアセンブラ機能と HAS060.X との互換性検証において完全な互換性を達成しました。
本計画は、コードベースの保守性、開発者体験（DX）、パフォーマンス、およびリリース運用の品質をさらに引き上げるための、今後の拡張ロードマップを定義します。

---

## 📅 ロードマップ概要

- **[x] 計画1**: 美しいエラーダイアグノスティック表示の導入（`miette` / `codespan-reporting` / カスタムインディケータ表示の組み込み完了）
- **[x] 計画2**: LSP (Language Server Protocol) サポートによるエディタ連携（完了）
- **[ ] 計画3**: 高速なハッシュマップと Perfect Hash Function (`phf`) の導入による高速化
- **[ ] 計画4**: ファイルパス表現の堅牢化（`PathBuf` や `OsString` の活用）
- **[x] 計画5**: クロスプラットフォーム自動リリース（GitHub Actions）の構築（完了）

---

## 🎨 計画1: 美しいエラー表示の導入 [x] 完了

### 目的
現在のプレーンテキストによるエラー報告から、モダンな Rust ツール（`rustc` など）のようにソースコード行を引用し、エラー位置をカラーでハイライトする視覚的な診断情報表示へ移行します。

### アプローチ
- `miette` または `codespan-reporting` クレートを導入します。
- `ErrorReporter` トレイトを拡張し、エラー発生行のバッファから該当部分を読み出し、視覚的なエラー矢印やヘルプメッセージを生成できるようにします。

---

## 💻 計画2: LSP (Language Server Protocol) サポート [x] 完了

### 目的
VSCode や他のエディタで `rhas` 向けの M68000 アセンブリ言語を書く際、リアルタイムのエラーチェック、自動補完、シンボル定義ジャンプなどを可能にし、レトロ開発の体験を劇的に向上させます。

### アプローチ
- `tower-lsp` 等のクレートを用い、`rhas --lsp` のようなサブコマンドで起動可能な LSP サーバー機能を統合します。
- エディタからの `didChange` / `didOpen` イベントに応じて Pass 1 のパーサーをインメモリで走らせ、Diagnostics（エラー位置情報）を返送します。

---

## ⚡ 計画3: シンボル検索と判定の高速化

### 目的
数万行を超える巨大なソースファイルをアセンブルする際の処理パフォーマンス（特にハッシュマップの検索速度）をさらに高速化します。

### アプローチ
- predefined な命令（Opcodes）やレジスタ名テーブルの検索に、コンパイル時に静的ハッシュを生成する `phf` クレートを導入します。
- `SymbolTable` や各解析フェーズで使用している `HashMap` に `rustc-hash` や `fxhash` などの高速なハッシュアルゴリズムを導入します。

---

## 🛡️ 計画4: ファイルパス表現の安全性の向上

### 目的
Windows、Linux、macOS などのクロスプラットフォーム環境において、日本語や特殊文字を含むマルチバイトファイルパス（SJIS, UTF-16等）を安全に処理し、インクルードファイル解決での不具合を完全に防ぎます。

### アプローチ
- パス表現を従来の `Vec<u8>` から、Rust標準の `std::ffi::OsString` および `std::path::PathBuf` へ移行します。
- `SourceStack` や `SourceBuf` におけるパス解決の記述を完全に型安全な API にリファクタリングします。

---

## 🚀 計画5: 自動リリースワークフローの構築 [x] 完了

### 目的
複数 OS 向け（Windows `x86_64`, Linux `x86_64`, macOS `Apple Silicon/x86_64`）のネイティブ実行バイナリを自動ビルドし、リリース時に即座に配布できるようにします。

### アプローチ
- GitHub Actions を利用して、タグプッシュ時に自動的に各プラットフォーム向けにクロスビルドを行うリリースジョブ（`release.yml`）を追加します。
- 各ターゲット環境向けにパッケージ化（zip/tar.gz）し、GitHub Release に自動アップロードするプロセスを自動化します。
