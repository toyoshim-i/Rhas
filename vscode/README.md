# Rhas X68000 Assembly Support

X68000 HAS アセンブラ（`.has` / `.s`）記述時の開発を強力に支援する VS Code 拡張機能です。内蔵された Rhas LSP サーバーにより、リッチな IDE 機能を提供します。

## ✨ 機能一覧

- 🎨 **シンタックスハイライト**
  - 行頭 `*` や `;` によるコメント、定数（`$`:16進数 / `%`:2進数 / `@`:8進数 / 10進数）、レジスタ、主要命令、疑似命令などを美しくカラーリングします。
- 🔍 **リアルタイムエラーチェック（Diagnostics）**
  - ファイル編集時にバックグラウンドでインメモリ・アセンブル（Pass 1〜3）を実行し、文法エラーや定義の重複、未定義シンボルなどをリアルタイムに赤・黄の波線でエディタ上にハイライトし、「問題（Diagnostics）」一覧に表示します。
- 📍 **定義へ移動（Go to Definition）**
  - ラベルや定数（`.equ` / `.set`）、マクロ（`.macro`）の参照部分で `F12` キーや `Ctrl+クリック` を押すことで、その定義行へ直接ジャンプできます。`.include` で読み込まれた別ファイル内の定義位置も正しく追跡してジャンプします。
- 💡 **ホバー表示（Hover）**
  - ラベルや定数にマウスを乗せると、値（10進数および16進数表記）や定義セクションなどをプレビューできます。マクロ名では仮引数一覧を、CPU命令やレジスタでは対応コードや仕様をホバー表示します。
- 🗂️ **アウトライン表示（Document Symbols）**
  - VS Code の「アウトライン」パネルに、ファイル内のラベル、定数、マクロなどを構造的に一覧表示します。長大なアセンブリファイルでもワンクリックでナビゲート可能です。
- ⌨️ **コード自動補完（Autocomplete / Completion）**
  - 68000の全命令、疑似命令（`.dc` / `.ds` 等）、およびファイル内で定義されたユーザーシンボルを入力候補として自動補完します。
- 💅 **自動フォーマッタ（Source Formatting）**
  - `Shift+Alt+F` または右クリックから「ドキュメントの整形」を実行することで、カラム（ラベル＝0列目、命令＝8列目、オペランド＝16/24列目、コメント）を美しく自動整列します。
- 🛠️ **アセンブル実行コマンド**
  - エディタを右クリックして「**Rhas: Assemble Current File**」を選択すると、バックグラウンドでアセンブルを実行し、ビルドログやエラーを VS Code の「Rhas Assembler」出力チャンネルへストリーム表示します（未保存の変更は実行前に自動で保存されます）。

## 🔧 導入手順

1. **拡張機能のインストール**
   GitHub Releases からリリース拡張機能ファイル `rhas-support-v3.0.9-x1.2.5-r2.vsix` をダウンロードしてインストールします。
   - **VS Code UI から**: 拡張機能ビュー（`Ctrl+Shift+X`）を開き、右上の `...`（三点リーダー）メニューから「VSIX からのインストール...（Install from VSIX...）」を選択し、ダウンロードした `.vsix` ファイルを指定します。
   - **コマンドラインから**: ターミナルで以下のコマンドを実行します。
     ```bash
     code --install-extension rhas-support-v3.0.9-x1.2.5-r2.vsix
     ```

2. **ゼロ構成（設定不要）での動作**
   拡張機能には、Windows、Linux、macOS 用のビルド済みアセンブラバイナリがすべて内包されています。
   **アセンブラ単体のインストールや実行パス（PATH）の設定を行う必要はなく、インストール後すぐにリアルタイムエラー表示が動作します。**

   *※特定のローカルバイナリパスの指定や、インクルードファイル（`.include`）の検索ディレクトリを設定したい場合は、VS Code の設定（`settings.json`）で以下を指定してください（相対パスはワークスペースのルートからのパスとして解決され、設定変更時はLSPサーバーが自動的に再起動します）：*
   ```json
   {
       // 実行バイナリの手動指定（通常は設定不要です）
       "rhas.executablePath": "/path/to/rhas",

       // インクルードファイルの検索ディレクトリ（複数指定可）
       "rhas.includePaths": [
           "libs",
           "include"
       ]
   }
   ```

> [!NOTE]
> **VS Code のプロファイル（Profiles）機能を使用している場合**
> デフォルト以外のカスタムプロファイルに拡張機能を明示的にインストールしたい場合は、`--profile` スイッチを指定してインストールを行ってください。
> ```bash
> code --profile "<プロファイル名>" --install-extension rhas-support-v3.0.9-x1.2.5-r2.vsix
> ```
